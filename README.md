# Overview

This is a base repo for building a SvelteKit application with the T3 stack and 3D graphics.
Included are the following pre-configured tools:

- SvelteKit
- TypeScript
- tRPC
- Tailwind CSS
- Spline
- Threlte (+ extras)
- Vite
- vite-imagetools
- Svelte Headless-UI
- Zod
- Prettier / ESLint
